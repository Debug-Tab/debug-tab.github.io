(超详细)使用Hexo+Github pages搭建免费美观的博客！

发表于2022-12-05|更新于2024-08-11
|阅读量:

0. 必要条件

  1. 一台能上网、没病毒的电脑
  2. 能折腾、有耐心的心
  3. 遇到报错等问题能使用正确的搜索引擎和关键字去查找答案
  4. 有基本的 电脑/网络 常识(经验)，如：电脑文件显示了后缀名、了解网页基本构造、知道cmd是什么、没安装XXX安全卫士/电脑管家

注: 接下来的内容如有不懂，请尝试暂时忽略

1. 基本介绍

Hexo

Hexo 是一款基于 Node.js(不懂没关系) 的静态(一会解释)博客框架，依赖少易于安装使用，可以方便地生成 静态 网页。

Node.js

Node.js 是一种 Javascript 运行环境，可以让 Javascript运行在本地(一般Javascript运行在浏览器上面)，而上面提到的 Hexo 基于Node.js

Github

Github 是全球最大的代码托管平台(你可以把开源的代码放在上面)，现已拥有超过900万开发者用户。

2. 安装软件

Node.js

注意：下载.msi的安装包，不要下载后缀.7z的压缩包(除非你知道自己在做什么)

此部分仅针对Windows用户，如果你用Linux，我相信你会安装软件。。。

方式1. 官网下载（简单，不推荐）

打开 https://nodejs.org/zh-cn/download/
直接点击你的系统版本就可以了，不懂的就点击这个(假设你是Windows x64)

方式2. 镜像下载（快）

打开 https://mirrors.tuna.tsinghua.edu.cn/nodejs-release
向下滑，不要滑到最底下，那是以前的版本，
目前最新的版本是v19.2.0，如果你害怕找错，可以直接进https://mirrors.tuna.tsinghua.edu.cn/nodejs-release/v19.2.0/

进入后向下划，下载这个（x86位用户请下载下面那个）

安装

运行下载的安装包，一路Next（可以修改下安装路径）

验证

安装时会默认配置环境变量
打开cmd（win+r 打开运行框，输入 cmd）
运行 node -v 和 npm -v ，不出意外的话你会看到这个，那恭喜你，安装成功了！

Git

打开https://git-scm.com/download/win ，最好选择安装包

如果你安装成功，打开命令行窗口，输入git –version ，就能看到输出的版本号了

Git Bash

Git Bash 是一种命令行，支持常用命令，甚至好像可以运行.sh
如果你安装时设置了右键Git bash(默认设置)的话，你就以后就可以右键文件夹空白区域，选择Git Bash Here来代替cmd。

注意！！！里面是不能Ctrl+V粘贴的，它会识别成两个不知道什么字符（如果你已经这样，按两下Backspace），不过幸好可以右键，按下Paste来粘贴。

3. 生成博客

创建一个文件夹，用来存放博客文件，在这里打开cmd

如果你不知道怎么做，先输入你的博客文件夹所在盘符，再输入 cd “你的路径” 如我的如下所示

如不出所料，你会看到如下画面：
